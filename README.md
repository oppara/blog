# [d.oppara.tv](https://d.oppara.tv)


## Usage

### create post with server

```console
% blog new
```
or

```console
% make
```

### edit post with server

```console
% blog edit
```

or

```console
% make edit
```

### start server

```console
% make up
```

### stop server

```console
% make down
```

### open browser

```console
% make open
```
